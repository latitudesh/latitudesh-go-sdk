# CreateServerReinstallServersRaid

RAID mode for the server. Set to 'raid-0' for RAID 0, 'raid-1' for RAID 1, or omit/null for no RAID configuration


## Values

| Name                                    | Value                                   |
| --------------------------------------- | --------------------------------------- |
| `CreateServerReinstallServersRaidRaid0` | raid-0                                  |
| `CreateServerReinstallServersRaidRaid1` | raid-1                                  |
