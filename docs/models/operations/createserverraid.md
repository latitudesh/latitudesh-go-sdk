# CreateServerRaid

RAID mode for the server. Set to 'raid-0' for RAID 0, 'raid-1' for RAID 1, or omit/null for no RAID configuration


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `CreateServerRaidRaid0` | raid-0                  |
| `CreateServerRaidRaid1` | raid-1                  |
